Boolean Model of a Protein-Protein Interaction (PPI)

Original Model

Encoding the state of interacting proteins to facilitate biological pathways reconstruction

Description

The interactions between proteins (e.g. phosphorylation, conformation, signal transduction, sub-cellular localization) are crucial for living cells. Understanding PPI's can provide some developments in not only systems biology but also novel therapeutics.

In the discrete model below, the interaction has been considered as phosphorylation of the residue "Ser 177" of a protein B by a protein A. In "Variables" and "Functions" below, x1 reflects the absence/presence of the protein A, whereas x2 indicates if the protein B has already been phosphorylated before the interaction. Right after this interaction, while f1 shows the absence/presence of the protein A, f2 points out whether the protein B has been phosphorylated or not. Since the interaction does not change any properties of the protein A, f1 must depend on only x1, in fact, it is equal to x1. On the other hand, the interaction does change some properties of the protein B, therefore f2 must depend on both x1 and x2.

Variables

x1 = Protein A
x2 = Protein B

Functions

f1 = x1
f2 = x1 + x2 + x1 * x2

Number of States

2 - Boolean

Key Dynamic Features

3 Components
3 Fixed points: 00, 01, 11
No 2-cycles, no 3-cycles
